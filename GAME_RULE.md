# 게임 이름: Catch Zone

## 1. 개요
* **Catch Zone**은 화면 상단에서 떨어지는 과일을 바구니로 받아내고, 폭탄을 피하는 포즈 인식 리듬/아케이드 게임입니다.
* 플레이어는 자신의 몸을 움직여 바구니를 이동시키고 점수를 획득합니다.

## 2. 구역 정의
* 화면은 가로로 3개의 구역으로 나뉩니다.
  * **LEFT (좌측)**
  * **CENTER (중앙)**
  * **RIGHT (우측)**

## 3. 조작 방식
* 웹캠을 통해 플레이어의 포즈를 인식하여 바구니 위치를 제어합니다.
  * **왼쪽 포즈 (Left Pose)** → 바구니가 **LEFT** 구역으로 이동
  * **중립 포즈 (Neutral Pose)** → 바구니가 **CENTER** 구역으로 이동
  * **오른쪽 포즈 (Right Pose)** → 바구니가 **RIGHT** 구역으로 이동

## 4. 아이템 종류
| 아이템 | 효과 | 비고 |
| :--- | :--- | :--- |
| **사과 (Apple)** | +100점 | 기본 과일 |
| **배 (Pear)** | +150점 | |
| **오렌지 (Orange)** | +200점 | |
| **폭탄 (Bomb)** | **즉시 게임 오버** | 피해야 함 |

## 5. 점수 규칙
* 아이템은 화면 상단에서 3개 구역 중 하나를 무작위로 선택하여 떨어집니다.
* 아이템이 바닥에 닿는 순간(혹은 판정 라인), 바구니가 해당 구역에 있다면 **획득**합니다.
* 바구니가 다른 구역에 있다면 **놓침(Miss)** 처리됩니다.
* 과일 획득 시 해당 점수만큼 총점이 증가합니다.

## 6. 게임 오버 조건
1. **폭탄 획득 시**: 즉시 게임 오버
2. **과일 놓침 (Miss) 누적 시**:
   * 놓침 1회: **경고 (Warning)**
   * 놓침 2회: **게임 오버 (Fail)**
   * (폭탄은 놓쳐도 상관없음, 획득했을 때만 문제됨)

## 7. 레벨 시스템
* **시간제 레벨업**: 각 단계(Level)는 **20초** 동안 진행됩니다.
* 20초 동안 게임 오버되지 않고 생존하면 자동으로 다음 레벨로 넘어갑니다.
* **난이도 상승 (Drop Time 감소)**
  * **1단계 Drop Time**: **2.0초** (화면 상단에서 바닥까지 도달 시간)
  * 레벨 증가 시: Drop Time이 **0.2초씩 감소** (더 빨라짐)
  * *예: 2단계 1.8초, 3단계 1.6초 ...*
* **아이템 생성 간격 (Spawn Interval)**
  * 현재 Drop Time의 **60% ~ 80%** 사이 무작위 시간 간격으로 아이템 생성

## 8. UI 요소
* **Score**: 현재 점수 표시
* **Level**: 현재 단계 표시
* **Time**: 다음 레벨까지 남은 시간 (20초 카운트다운) 또는 전체 생존 시간
* **Miss Count**: 과일 놓친 횟수 (0/2, 1/2 표시)
* **Game Over Overlay**: 게임 종료 시 최종 점수와 재시작 버튼 표시
