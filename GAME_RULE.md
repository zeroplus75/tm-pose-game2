# 게임 이름: Catch Zone

## 기본 구조:
* 화면은 LEFT / CENTER / RIGHT 3개 구역으로 구성된다.
* 플레이어는 포즈 인식으로 바구니를 조작한다.
  * 왼쪽 포즈 → LEFT
  * 중립 포즈 → CENTER
  * 오른쪽 포즈 → RIGHT

## 아이템 종류:
1. 폭탄(Bomb) → 닿으면 즉시 게임 오버
2. 사과(Apple) → +100점
3. 배(Pear) → +150점
4. 오렌지(Orange) → +200점

## 그 외:
* 과일 놓침 2번 시 게임 오버
* 20초마다 레벨 증가 (아이템 속도 증가)
